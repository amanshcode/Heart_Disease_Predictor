# ❤️ Heart Disease Prediction App

A Machine Learning-powered web application that predicts the likelihood of heart disease based on various clinical parameters. The application provides an intuitive interface built with Streamlit, allowing users to enter patient details and receive an instant prediction.

## 🚀 Live Demo

🔗 **Demo:** `[https://your-streamlit-app-url.streamlit.app](https://heart-disease-predictor-git-xv.streamlit.app/#heart-disease-prediction-app)`


---

## 📌 Features

* 🩺 Predicts the risk of heart disease in real time.
* 📊 Interactive and user-friendly Streamlit interface.
* 📘 Built-in feature guide explaining medical abbreviations.
* ⚡ Fast predictions using a trained K-Nearest Neighbors (KNN) model.
* 🔍 Data preprocessing with feature scaling and one-hot encoding.

## 📊 Model Used

**Algorithm:** K-Nearest Neighbors (KNN)

The model was trained on heart disease data after preprocessing and feature scaling. User inputs are transformed using the same preprocessing pipeline before prediction to ensure consistency.

---

## 📋 Input Features

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise-Induced Angina
* Oldpeak (ST Depression)
* ST Slope
